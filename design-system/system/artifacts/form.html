Fellowship Focus — app Windows de productivité pour freelances et devs qui bossent en guild (accountability sociale). Mission : bloquer les distractions (Twitter, YouTube Shorts, TikTok) pendant des sessions focus 45 min, tracker XP/habits/OKRs avec des amis — sans surveillance agressive. Voix : calme, professionnelle, minimaliste. Pas fantasy RPG, pas gold partout. Think Linear × Opal × Notion — dark, clean, data-first. Utilisateur : dev/freelance 25–40 ans, Windows, veut un dashboard qui respire, pas un jeu vidéo. Interdits UI : JSON/token visible, debug text, hero banners vides, double sidebar, serif sur les chiffres, boutons gold gradient pleine largeur. Références visuelles : Linear app, Opal, Raycast, Vercel dashboard — pas Lord of the Rings UI kits.
