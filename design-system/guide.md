# Fellowship Focus — Brand Guide

**Focus en guild. Sans distraction.**

App Windows de productivité pour freelances et devs en guild. Sessions focus 45 min, blocker système, XP / habits / OKRs — esthétique Linear, pas jeu vidéo.

## Couleurs

| Rôle | Hex | Usage |
| --- | --- | --- |
| Background | `#1a1c1e` | canvas, sidebar |
| Text | `#f4f4f5` | titres, corps, KPI |
| Accent | `#b8422e` | CTA, nav active, anneau timer |
| Surface | `#242628` | cards |
| Muted | `#9ca3af` | meta, nav inactive |
| Border | `#3a3d40` | filets 1px |
| Success | `#2d6a4f` | états OK |

## Typo

- **Cinzel** — wordmark uniquement (11px, letter-spacing large, muted)
- **Public Sans** — tout le reste, y compris les chiffres

## Layout produit

- Sidebar 200px · top bar 48px · content max 960px
- Radius 6–10px · bordures 1px · base spacing 4px
- Jamais : gold, fantasy, hero banner vide, JSON visible
