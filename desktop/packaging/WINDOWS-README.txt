Fellowship Focus — Windows

1. Unzip this folder anywhere (e.g. Desktop\FellowshipFocus)
2. Double-click "Start Fellowship Focus.bat"
   OR open FellowshipFocus.exe directly
3. First run: Blocker tab → Generate + Install certificate (once)
4. Settings tab → paste your invite link from the web dashboard

Do NOT run python main.py — use FellowshipFocus.exe only.
Keep the whole folder together (_internal contains required libraries).

Support: https://fellowship-focus-production.up.railway.app/download
